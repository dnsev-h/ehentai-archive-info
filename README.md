# E*Hentai Archive Info

Add info.json files to archives.

Prerequisites:
* [7zip](https://www.7-zip.org/)
* [nodejs](https://nodejs.org/) (node and nmp required)

First time setup:
* Run ```setup.bat```

Updating archives:
* Drag and drop archive files onto ```batch.bat```
